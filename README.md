## Medical Insurance Cost Predictor (Regression App) ##
* A Streamlit app that predicts the estimated annual medical insurance cost based on personal details using a trained Random Forest regression model

### Live demo ###
Check out the app here: [Medical Insurance Cost Predictor on Streamlit] (https://insurance-regression.streamlit.app/)

### How it works ###
* Inputing features: age, BMI, number of children, sex, smoker status and region allows the app which uses a trained Random Forest regression model to predict the expected annual medical insurance cost for the user
